# Kita Growth Engine

Interactive growth model for Kita (YC W26).

## Deploy to Vercel

### Option A: From this folder
1. `npm install`
2. `npm run dev` (to preview locally at localhost:5173)
3. Push to GitHub
4. Import the repo in Vercel â€” it auto-detects Vite
5. Deploy

### Option B: Vercel CLI
1. `npm install`
2. `npx vercel` (follow prompts)

### Settings Vercel should auto-detect:
- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`

No environment variables needed.
